# Learn Core of PHP Framework

Will try to make updates so keep an eye.

## Installation

### Requirements

PHP >= 7.0.0

### USE
Clone the repository
```bash
git clone https://github.com/AhmedRaafat14/corephp-framework.git
```

Then run composer install
```bash
cd corephp-framework

composer install
```
